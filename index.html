To use:

Drag elements from the objects tray on the left into the middle stage diagram to add an object to the stage diagram. Double click or click the edit icon to edit the input number of a selected object. Clicking the trash icon will delete all selected objects. Clicking the duplicate icon will duplicate all selected objects. Clicking the clear canvas icon will delete all objects. Clicking the save icon will save the title, stage diagram, and notes to a PDF to be printed or emailed. Clicking the notes icon will allow you to write down any notes that could be pertinent to the stage diagram.
